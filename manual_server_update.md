# 手动更新服务器上的应用

由于SSH认证问题，需要手动更新服务器上的应用。请按以下步骤操作：

## 方法一：直接在服务器上更新（推荐）

1. 在您的本地计算机上，使用SSH连接到服务器：
   ```bash
   ssh admin@8.137.39.61
   ```

2. 进入应用目录：
   ```bash
   cd /home/admin/fuel-tracker-app
   ```

3. 拉取最新的代码更改：
   ```bash
   git pull origin main
   ```

4. 重启应用服务：
   ```bash
   pkill -f gunicorn
   /home/admin/.local/bin/gunicorn -w 1 -b 0.0.0.0:8000 app:app &
   ```

## 方法二：如果您无法SSH登录服务器

如果您无法通过SSH登录服务器，请联系服务器管理员完成以下操作：
- 拉取最新的代码更改
- 重启Gunicorn服务

## 验证更新

更新完成后，访问 http://8.137.39.61 验证删除功能是否正常工作：
- 检查加油记录表格中是否有"删除"按钮
- 尝试删除一条记录，确认功能正常

## 功能说明

此次更新添加了燃油追踪应用的删除功能：
- 在记录表格中添加了删除按钮
- 添加了删除确认对话框
- 实现了后端删除API接口
- 用户可以删除不需要的加油记录
